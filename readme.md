# shooting-game

射击小游戏

## 项目说明

- 概述：Canvas 射击小游戏要求玩家控制飞机发射子弹，消灭会移动的怪兽，如果全部消灭了则游戏成功，如果怪兽移动到底部则游戏失败.

## 项目结构

- **index.html**: 游戏页面
- **style.css**: 页面样式
- **js**: 页面涉及的所有 js 代码
- **app.js**: 页面逻辑入口
- **img**: 存放游戏的图片素材
- **readme.md**: 项目说明文档
